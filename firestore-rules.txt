# Regras do Firestore para O Irmãozinho

## Regras Temporárias para Teste (PERMISSIVAS - APENAS PARA DESENVOLVIMENTO)

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Permitir leitura e escrita para usuários autenticados
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
```

## Regras de Produção (RECOMENDADAS)

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Coleção de conteúdo - apenas usuários autenticados
    match /content/{document} {
      allow read, write: if request.auth != null;
    }
    
    // Coleção de teste - apenas para desenvolvimento
    match /test/{document} {
      allow read, write: if request.auth != null;
    }
    
    // Bloquear acesso a outras coleções
    match /{document=**} {
      allow read, write: if false;
    }
  }
}
```

## Como Configurar no Firebase Console:

1. Acesse: https://console.firebase.google.com/
2. Selecione o projeto: admoirmaozinho
3. Vá em: Firestore Database > Rules
4. Substitua as regras atuais pelas regras acima
5. Clique em "Publish"

## Verificações Importantes:

1. **Firestore está habilitado?**
   - Firestore Database > Data > Verificar se existe

2. **Modo de produção ou teste?**
   - Se estiver em modo de produção, mude para modo de teste temporariamente

3. **Regras estão configuradas?**
   - As regras padrão bloqueiam tudo
   - Configure as regras permissivas para teste

4. **Projeto está ativo?**
   - Verifique se o projeto não está pausado ou suspenso
